__What is Data Structure?__

a data structure is a way to organize data in a form that is accessible to computers

+ Types of Data Structure
  - Arrays
  - Linked Lists
  - Stacks
  - Queues
  - Hash Tables
  - Trees
  - Heaps
  - Graphs
